# Particle Functions
 Particle functions allow you to build and spawn particles.

## spawnParticles

<h4 style="padding-top: 4.6rem"> Signature: </h4>

```js
Boolean spawnParticles(ParticleEffect particleEffect, Location location, Viewer viewer, Integer radius)
```
